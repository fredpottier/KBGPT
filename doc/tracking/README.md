# Suivi (Tracking)

Ce répertoire contient les documents de **statut**, de planification et de backlog.

## Repères rapides

- `OSMOSE_STATUS_ACTUEL.md` : état global projet
- `OSMOSE_PHASE2_TRACKING.md` : suivi Phase 2
- `OSMOSIS_EXTRACTION_V2_TRACKING.md` : suivi extraction V2
- `BACKLOG_IDEAS.md` / `BACKLOG_BURST_IMPROVEMENTS.md` : idées et améliorations

## Règles

- Le tracking reflète **l'état réel**.
- Les specs vivent dans `doc/specs/`.
- Les analyses et comparatifs vivent dans `doc/research/`.
